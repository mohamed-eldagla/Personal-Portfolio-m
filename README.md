## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎬 **Scroll Animations** | Staggered entrance animations with slide-left, slide-right, scale, and fade-up variants |
| 🖼️ **Project Thumbnails** | AI-generated preview images with hover zoom effects |
| 🏢 **Org Logos** | Real university & company logos in Education and Experience sections |
| 🌗 **Dark / Light Mode** | Seamless theme switching with full design parity |
| 📱 **Fully Responsive** | Optimized for desktop, tablet, and mobile viewports |
| 🎨 **Glassmorphism UI** | Modern glass-card design with subtle gradients and ambient glows |

## 📂 Project Structure

```
src/
├── assets/
│   ├── logos/          # University & company logos (GIU, E-JUST, IDSC, UPEC, UN, DataScience ME)
│   └── projects/       # AI-generated project thumbnail images
├── components/
│   ├── AnimatedSection.tsx    # Reusable scroll-triggered animation wrapper
│   ├── HeroSection.tsx        # Landing hero with typing effect
│   ├── AboutSection.tsx       # Bio, stats, skills, and languages
│   ├── ExperienceSection.tsx  # Timeline with company logos
│   ├── ProjectsSection.tsx    # Project cards with thumbnails
│   ├── EducationSection.tsx   # Education cards with university logos
│   ├── PublicationsSection.tsx
│   ├── ExtracurricularSection.tsx
│   ├── AwardsSection.tsx
│   ├── ContactSection.tsx
│   └── Navbar.tsx
├── pages/
│   └── Index.tsx              # Main page layout
└── index.css                  # Global styles & design tokens
```

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/mohamed-eldagla/Personal-Portfolio-m.git
cd Personal-Portfolio-m

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
