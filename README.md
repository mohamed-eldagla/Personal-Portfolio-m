# Portfolio Spark

A modern, responsive portfolio built with:
- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## Getting Started

First, install the necessary dependencies:

```sh
npm install
```

Start the development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```
